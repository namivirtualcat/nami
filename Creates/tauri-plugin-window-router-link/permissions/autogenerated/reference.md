## Default Permission

This permission set configures what kind of
operations are available from the mcp plugin.

#### Granted Permissions

All operations are enabled by default.

#### This default permission set includes the following:

- `allow-go`

## Permission Table

<table>
<tr>

<th>Description</th>
</tr>


<tr>
<td>
